\section{Shiny App}\label{shiny-app}}

The three main components of a Shiny App are the input, output, and the
user interface (UI). \emph{Input} is comprised of input controls which
allow a user to interact with reactive functions. \emph{Output} is
comprised of objects that display different visualizations based on the
type of input provided. These can be as simple as text or as complicated
as an interactive plot. The \emph{UI} is comprised of a user interface
\emph{object} and the \emph{server} that it communicates with.They work
as a pair.
